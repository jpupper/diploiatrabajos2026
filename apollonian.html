Explicación: La Junta de Apolonio (Apollonian Gasket) se genera empacando infinitos círculos mutuamente tangentes. Este fractal se calcula resolviendo recursivamente el Teorema de Círculos de Descartes, el cual relaciona las curvaturas ($k_i = 1/r_i$) de cuatro círculos tangentes:

(k₁ + k₂ + k₃ + k₄)² = 2(k₁² + k₂² + k₃² + k₄²)

Mover el mouse horizontalmente altera el radio de los primeros círculos internos tangentes. A partir de esa variación, el resolvedor ejecuta Descartes en tiempo real recalculando las posiciones complejas de toda la jerarquía de círculos, provocando una danza matemática perfecta donde ningún círculo se superpone ni pierde tangencia.
